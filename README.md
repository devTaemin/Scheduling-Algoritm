<br>

###  📌 List of Scheduling algorithm
-----------------
###### 📌FCFS: First Come First Served (Non-preemptive)
###### 📌SJF: Shortest Job First (Non-preemptive)
###### 📌SRJF: Shortest Remaining Job First (Preemptive)
###### 📌RR: Round Robin (Preemptive)
